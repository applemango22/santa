02  핵심 기능
🔒
은행 로그인 없음
은행 계좌 연동 없이 수동 입력만으로 구독을 추적합니다. 금융 자격증명을 제3자와 공유하지 않으며, PIPEDA 개인정보보호 표준을 준수합니다.
🤖
AI 지출 분석
AI가 전체 구독 목록을 분석해 최대 지출 항목, 카테고리별 소비 패턴 등 사용자가 미처 인지하지 못한 낭비 구독을 찾아줍니다.
⏱️
무료체험 카운트다운
진행 중인 무료 체험의 남은 일수를 색상으로 구분해 표시하고, 유료 전환 3일 전 자동 알림을 전송합니다.
🔔
갱신 리마인더
구독 갱신일 3일 전 푸시 알림을 발송합니다. 각 구독마다 알림 타이밍을 개별 설정할 수 있습니다.
📦
100+ 서비스 내장
Netflix, Spotify, iCloud, ChatGPT, Adobe 등 100개 이상의 구독 서비스가 사전 등록되어 있어 빠른 추가가 가능합니다.
📴
오프라인 · 로컬 우선
인터넷 없이도 완전히 동작하며 데이터는 기기에 로컬 저장됩니다. Face ID / Touch ID 앱 잠금 기능도 제공합니다.

04  잘한 점 & 개선점
✅
잘한 점
명확한 차별점 — "No Bank Login" 경쟁 서비스(Mint, 뱅크샐러드 등)가 금융 연동에 집중하는 반면, 은행 로그인 불필요를 전면에 내세워 프라이버시 민감층을 정확히 공략합니다.
슬로건이 즉시 이해되는 가치 전달 페이지 타이틀 한 줄에 "무엇을(구독 추적) + 어떻게(은행 로그인 없이) + 추가 장점(AI, 무료)"을 모두 담아 첫인상이 선명합니다.
완전 무료 진입 + 로컬 스토리지 계정 생성 없이 즉시 사용 가능하고, 오프라인·로컬 저장으로 사용 허들이 최소화되어 전환율에 유리합니다.
무료체험 카운트다운 — 실용적 기능 무료체험을 잊고 유료 전환되는 흔한 고통을 정확히 해결합니다. 색상 경고 + 3일 전 알림 조합이 실효성이 높습니다.
생체인증 보안 (Face ID / Touch ID) 금융 정보 앱임에도 복잡한 로그인 없이 보안을 제공해, 편의성과 보안 둘 다 만족시킵니다.
💡
개선점
SPA로 인한 SEO 취약성 JavaScript 렌더링 의존 구조로 검색 크롤러가 본문을 수집하지 못합니다. SSR(Next.js) 또는 정적 사전 렌더링 도입이 필요합니다.
한국(.kr) 도메인인데 한국어 지원 불분명 .kr 도메인 사용에도 불구하고 App Store 설명이 전부 영어입니다. 한국 사용자를 실제 타겟으로 한다면 로컬라이제이션이 필요합니다.
3단계 Pro 가격 구조가 복잡 $3.99 / $9.99 / $49.99 세 가지 티어는 단순 구독 추적 앱치고 선택 피로를 유발합니다. 단일 Pro 플랜으로 단순화가 권장됩니다.
수동 입력의 진입 장벽 은행 연동 없이 수동 입력 방식은 프라이버시 장점이지만, 첫 설정 수고가 높습니다. 이메일 영수증 스캔 또는 구독 자동 감지 보조 기능이 필요합니다.
소셜 증거(리뷰·사용자 수) 부재 App Store에 리뷰가 충분하지 않고 웹사이트에서도 사용자 수/후기 노출이 부족합니다. 신뢰도 확보를 위한 사회적 증거 강화가 필요합니다.
05  총평
⭐⭐⭐⭐
SubTrackr(subtrac.kr)는 "은행 로그인 없이 구독을 추적한다"는 명확하고 차별화된 포지셔닝을 가진 개인 재무 관리 앱입니다. 특히 핀테크 서비스에 금융 계정 연동을 꺼리는 사용자층을 정확히 겨냥한 점, AI 분석과 무료체험 카운트다운 같은 실용적 기능 조합, 무료·계정 불필요 전략이 돋보입니다.

다만 SPA 구조로 인한 SEO 약점, 한국 도메인 대비 한국어 미지원, 수동 입력의 불편함은 성장을 저해하는 요인입니다. 이메일 기반 구독 감지, SSR 도입, 한국어 현지화 세 가지를 우선 개선한다면 한국 시장에서 경쟁력 있는 서비스로 자리잡을 가능성이 높습니다.
